# Card Deck Assignment

this is my java project for drawing cards and showing them in a window. i have 2 files in the package com.java.assignment.card_deck.

### files:
1. Card.java - this is the logic for the cards. it has enums for Suit (hearts, diamonds etc) and Rank (ace to king). it also has the Deck class where it makes 52 cards and has a shuffle method. i used a list for the cards.
2. Main.java - this is the main class. it prints 20 cards at the start and then opens a swing window. it has the animation where the card moves from the deck to the table. 

### features:
- **Draw**: draws 1 card with animation. i used a timer for this.
- **Draw 10**: draws 10 cards fast if you dont want to wait for animation.
- **Shuffle**: resets the deck and shuffles everything.
- **Sort**: sorts the drawn cards by suit value and then rank.
